Como criar um autenticador de conexão PPPoE personalizado:
(versão gratuita)

Aqui explicaremos como criar o autenticador personalizado para seu provedor, começaremos com a instalação do programa necessário para a criação do instalador, o Inno Setup Compiler, segue o link:

Inno Setup QuickStart Pack 5.3.9

A instalação do Inno Setup QuickStart Pack é bem simples, só ir avançando…



Caso queira adicionar mais opções ao programa de instalação do autenticador, você pode instalar também o ISTool, que é um programa que auxilia na criação de scripts para o Inno Setup Compiler. Neste tutorial não iremos trabalhar com ISTool, mas caso queira instalar marque a opção "Download and install ISTool" conforme na imagem abaixo.



A próxima tela irá perguntar se você deseja instalar o Inno Setup Preprocessor, deixe marcada a opção "Install Inno Setup Preprocessor", pois para poder compilar o instalador precisaremos deste add-on.



Depois virá a tela perguntando se você gostaria de instalar o suporte a encriptação, deixe marcada a opção "Download and install encryption support", agora é só ir avançando…



Após a instalação Inno Setup QuickStart Pack, baixe o arquivo compactado que contém todos os arquivos necessários para que você possa fazer a compilação do instalador, segue o link:

Arquivos do Instalador do autenticador de conexão PPPoE (não compilado)

Após feito o download, descompacte o arquivo em algum local de sua preferência. Caso você não tenha o WinRAR para descompactar o arquivo, segue o link para download:

WinRAR 3.93

Após descompactado, abra o Inno Setup Compiler, se aparecer a janela "Welcome" você clique no botão "Cancel", após isso clique no menu "File" > "Open..."



Procure a pasta que você descompactou e abra o arquivo script.iss, após abrir o arquivo, procure as seguintes linhas:

#define nome "Autenticador Provedor NET.BR"
#define empresa "NET.BR Serviços de Internet"
#define url "http://www.provedor.net.br/"
#define logo "logo.png"
#define icone "icone.ico"

Agora altere as linhas, colocando os seus dados. Lembrando que os dados devem estar entre as aspas para que não haja falha na compilação. Vamos especificar o que cada variável representa:

nome = Nome do autenticador, o instalador terá parte deste nome e o executável do autenticador.

empresa = Empresa que compilou o instalador, sua empresa.

url = Endereço para qual o cliente será redirecionado ao se conectar pelo autenticador.

logo = É o nome da imagem que o autenticador terá, deverá estar na mesma pasta onde foi descompactado os arquivos, recomendamos que o tamanho da imagem seja 330x115 pixels no formato .bmp, .gif, .jpg, .png ou .tif, caso queira a imagem com transparência, use o formato .png.

INFORMAÇÂO: Quando você baixa o arquivo para compilação, dentro dele tem um arquivo de demostração do formato do logo, com o nome logo.psd, pode ser aberto pelo programa Photoshop.

icone = Nome do arquivo de ícone do autenticador, deverá estar na mesma pasta onde foi descompactado os arquivos, vem com um ícone padrão do autenticador, mas pode ser alterado, basta você criar um ícone com 3 tamanhos dentro do mesmo arquivo, os tamanhos são: 16x16, 32x32 e 48x48 pixels, deve estar com a extenção .ico, recomendamos o programa MicroAngelo Toolset para edição dos ícones, segue o link para download do programa (trial):

MicroAngelo Toolset 6.10.0070

Caso tenha conhecimento do software Inno Setup Compiler, pode editar as opções para personalizar a instalação do autenticador, após terminar a edição, salve o arquivo, depois clique no menu "Run" > "Run" ou precione "F9"



Pronto, o Inno Setup Compiler irá compilar o instalador na mesma pasta dos arquivos descompactados, segue o link para o arquivo de demonstração compilado do instalador.

Instalador do autenticador de conexão PPPoE (compilado)
